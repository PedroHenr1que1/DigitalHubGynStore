# Henrique Cell - Digital Hub

Este é o hub digital da Henrique Cell, apresentando smartphones, acessórios e serviços.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Shadcn/UI
- PNPM (Gerenciador de pacotes)

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   pnpm install
   ```

2. Rode o servidor de desenvolvimento:
   ```bash
   pnpm dev
   ```
